# HIPT checkpoints

HIPT pretrained checkpoints are not distributed with HiCAT.

Please download the official HIPT checkpoints before running `model="hipt"`.

Expected files:

```text
hicat_spatial/preprocessing/hipt/checkpoints/vit256_small_dino.pth
hicat_spatial/preprocessing/hipt/checkpoints/vit4k_xs_dino.pth
```

Alternatively, pass a directory containing both files as `checkpoint_path`.
