# HIPT checkpoints

HIPT pretrained checkpoints are not distributed with HiCAT.

Please download the official HIPT checkpoints before running `model="hipt"`.

Expected files:

```text
hicat/preprocessing/hipt/checkpoints/vit256_small_dino.pth
hicat/preprocessing/hipt/checkpoints/vit4k_xs_dino.pth

bash tutorial/download_hipt_checkpoints.sh hicat/preprocessing/hipt/checkpoints
